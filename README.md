# WindowSeat

An intelligent full-stack application that recommends optimal flight seats based on real-time weather conditions, scenic landmarks, and AI analysis.

## Features
- **Smart Seat Recommendations**: AI-powered analysis considering weather, landmarks, and flight paths
- **Real-time Weather Data**: Live weather conditions along your flight route
- **Landmark Detection**: Automatically identifies scenic views and points of interest
- **Confidence Scoring**: Transparent confidence levels for each recommendation
- **RAG Enhancement**: Learns from historical flight data to improve accuracy over time
